### Installation
* install nodejs from [official websete](https://nodejs.org/en/download/) 
* clone respositry
  ```bash
  $ git clone https://github.com/rahulsa123/blog_frontend.git
  $ cd blog_frontend
  ```
* Install dependencies and run server
  ```bash
  $ npm i
  $ npm start
